## 4.2 齐次线性方程组

基础解系

---

【总结】线性方程组解的结构必背结论

关联 660 题 - 321题

1. 设 $x_1, x_2, \cdots, x_p$ 是 $Ax=0$ 的 $p$ 个解，则 $k_1 x_1 + k_2 x_2 + \cdots + k_p x_p$ 也是 $Ax=0$ 的解；
2. 设 $x_1, x_2, \cdots, x_p$ 是 $Ax=0$ 的 $p$ 个解，则 $k_1 x_1 + k_2 x_2 + \cdots + k_p x_p$ 是 $Ax=0$ 的解，其中 $k_1 + k_2 + \cdots + k_p = 0$；
3. 设 $x_1, x_2, \cdots, x_p$ 是 $Ax=b$ 的 $p$ 个解，则 $k_1 x_1 + k_2 x_2 + \cdots + k_p x_p$ 是 $Ax=b$ 的解，其中 $k_1 + k_2 + \cdots + k_p = 1$；
4. 设 $x$ 是 $Ax=0$ 的一个解，$y$ 是 $Ax=b$ 的一个解，则 $x+y$ 是 $Ax=b$ 的一个解；
5. 设 $A$ 是 $m \times n$ 矩阵，$r(A) = r$，且 $x_1, x_2, \cdots, x_{n-r}$ 是 $Ax=0$ 的 $n-r$ 个线性无关的解，则 $x = k_1 x_1 + k_2 x_2 + \cdots + k_{n-r} x_{n-r}$ 也是 $Ax=0$ 的通解。
